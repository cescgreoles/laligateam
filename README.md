## Next

Execute while in development:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Execute in production:

```bash
npm run start
```

## Database

Every time you change the schemas, you need to run the following command to update the database:

```bash
npm run db:push
```

To see the database schema, run:

```bash
npm run db:studio
```
